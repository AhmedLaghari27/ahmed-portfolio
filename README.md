# ⚡ Ahmed Laghari — Personal Portfolio

> A futuristic, dark-themed developer portfolio built with **Vite + React** — featuring neon aesthetics, glitch effects, typewriter animations, scroll reveals, and a custom cursor experience.

## ✨ Features

- 🎯 **Glitch effect** on hero name — cyberpunk-style flicker
- ⌨️ **Typewriter animation** cycling through roles
- 🖱️ **Custom neon cursor** with a smooth trailing ring
- 🔲 **Animated grid background** with a live scan line
- 🎞️ **Scroll reveal animations** — sections fade up on enter
- 💡 **Neon hover interactions** on every card, button, and link
- 🌐 **Fully responsive** across all screen sizes
- 🚀 **Zero external animation libraries** — pure CSS animations

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/AhmedLaghari27/portfolio.git

# 2. Navigate into the project
cd portfolio

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

The app will be running at `http://localhost:5173`

### Build for Production

```bash
npm run build
```
